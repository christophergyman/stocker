📈 **Project Overview: Stocker - React Stock Trading App**

**Core Setup & Architecture:**
- Vanilla React + TypeScript project with Tailwind CSS
- Custom color palette: cream (#FFFCE2), beige (#CCC5B9), charcoal (#403D39), dark (#252422), terracotta (#EB5E28)
- Custom font: "Doto" monospaced dotted font
- React Router v6 for navigation between pages

**Component Structure:**
src/
├── components/
│   ├── Header.tsx          ← Stock emoji + "Stocker." text
│   ├── HomeButton.tsx      ← Home navigation + animated underline
│   ├── MicroButton.tsx     ← Micro page navigation + animated underline
│   └── SearchButton.tsx    ← Search functionality + animated underline
├── pages/
│   ├── HomePage.tsx        ← Main landing page (/)
│   └── MicroPage.tsx       ← Micro trading page (/micro)
└── App.tsx                 ← Router setup

**Key Features Implemented:**

🎨 Design System:
- Dark background (#252422) with cream/terracotta accents
- Custom Tailwind color classes (bg-cream, text-terracotta, etc.)
- Doto font family for all text elements
- Responsive grid layout (3 columns on desktop, 1 on mobile)

🚀 Navigation & Routing:
- Home button: Navigates to "/" + page refresh
- Micro button: Navigates to "/micro" page
- Search button: Ready for search functionality
- React Router with BrowserRouter and Routes

✨ Interactive Elements:
- Animated dotted underlines on all buttons (hover-triggered)
- Hover effects: Text color changes from cream to terracotta
- Smooth transitions: 200ms color transitions
- Animation timing: 0.3s hover, 0.2s reverse

🎯 UI Components:
- Header: Stock emoji (📈) + "Stocker." text in top-left
- Centered button grid: 3 main navigation buttons
- Back navigation: From Micro page to Home
- Browser favicon: Stock emoji (📈) in tab

**Technical Implementation Details:**

CSS Animations:
@keyframes drawLine {
  0% { opacity: 0; transform: scale(0); }
  50% { opacity: 1; transform: scale(1.2); }
  100% { opacity: 1; transform: scale(1); }
}

@keyframes drawLineReverse {
  0% { opacity: 1; transform: scale(1); }
  100% { opacity: 0; transform: scale(0); }
}

Button Animation Logic:
- 8 dots with sequential animation delays
- Hover state management with useState
- Conditional animation based on isHovered state
- Reverse animation when mouse leaves

Routing Structure:
<BrowserRouter>
  <Routes>
    <Route path="/" element={<HomePage />} />
    <Route path="/micro" element={<MicroPage />} />
  </Routes>
</BrowserRouter>
